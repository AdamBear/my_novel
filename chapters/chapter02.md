# 第2章：完美攻略

## 陷阱设置

林逸握着消防斧，站在三楼的走廊里。

虚拟地图上显示，还有10个红点分布在教学楼各处。他已经击杀了第一只丧尸，现在他需要执行他的"完美计划"。

"先清理通往图书馆路上的敌人，然后设置陷阱。"

他打开背包，检查物品：创可贴×2、急救包×1、能量饮料×1。

"补给充足。"

林逸开始行动。

他沿着走廊前进，来到第一个陷阱点——楼梯口。

这里是必经之路，如果有丧尸追击，一定会经过这里。

林逸推来几张桌子，堵住楼梯的一侧，只留下一个狭窄的通道。然后他把几把椅子倒放在通道里，椅子腿朝上。

"这样可以延缓它们的速度。"

他在椅子上洒了一些碎玻璃——从破碎的窗户上收集的。

"如果它们踩上去，会发出声音，我就能提前知道。"

第一个陷阱完成。

林逸继续前进，来到二楼。

虚拟地图显示，二楼有3个红点。他需要清理这些敌人，确保通往图书馆的路线安全。

他小心翼翼地走着，脚步很轻。

第一只丧尸在音乐教室附近游荡。

林逸没有直接攻击，而是先观察周围的环境。

音乐教室的门半开着，里面有钢琴、乐谱架。走廊的墙边有几个垃圾桶。

"可以利用。"

林逸悄悄走到垃圾桶旁边，用力踢了一脚。

垃圾桶发出巨大的声响，滚到走廊中央。

丧尸立刻被吸引过来，朝垃圾桶的方向走去。

林逸绕到它的背后，趁它注意力在垃圾桶上时，挥斧攻击。

一击命中头部。

丧尸倒地，消失。

```
【击杀记录】
丧尸学生 ×2
经验值：+20
```

"完美。"

**爽点1：声东击西，潜行击杀**

林逸感到一阵成就感。这种利用环境、智取敌人的感觉，比正面硬刚要爽快得多。

他继续前进，寻找下一个目标。

第二只丧尸在走廊尽头。

这次林逸用了不同的战术。他把一把椅子放在走廊中间，然后躲在拐角处。

丧尸发现椅子，走过去查看。

就在它低头的瞬间，林逸冲出来，一斧砍在它的后脑勺上。

又是一击必杀。

```
【击杀记录】
丧尸学生 ×3
经验值：+30
```

**爽点2：陷阱诱杀**

"这游戏的AI还是太简单了。"

林逸越来越自信。他发现只要利用环境和战术，这些丧尸根本不是威胁。

第三只丧尸在楼梯口附近。

这次林逸更大胆了。他直接走到丧尸面前，等它扑过来时，侧身躲避，然后反击。

整个过程行云流水，没有任何失误。

```
【击杀记录】
丧尸学生 ×4
经验值：+40
```

**爽点3：正面对决，完美闪避**

林逸看了看倒计时：18分32秒。

"二楼清理完毕，用时不到7分钟。"

他在心里计算着。按照这个速度，他可以在15分钟内清理所有敌人，然后进入图书馆，轻松通关。

"这个攻略视频肯定能火。"

林逸继续设置陷阱。

他在二楼的走廊里放置了更多障碍物——桌子、椅子、垃圾桶。如果有丧尸追过来，这些障碍物会大大延缓它们的速度。

他还在楼梯口撒了更多碎玻璃，制造"警报系统"。

"完美的防御体系。"

林逸满意地点点头，准备前往一楼。

---

## 连续击杀

一楼的敌人更多。虚拟地图显示有5个红点。

林逸没有慌张。他已经找到了对付这些丧尸的诀窍——利用环境、保持距离、一击必杀。

他来到一楼的大厅。

大厅很宽敞，有几根柱子，还有一些长凳。光线比楼上好一些，从大门透进来的阳光照亮了地面。

三只丧尸在大厅里游荡。

"三只...可以一起解决。"

林逸观察着它们的位置。三只丧尸分散在大厅的不同角落，距离都不近。

"如果能把它们引到一起，然后利用柱子作为掩护..."

林逸开始执行计划。

他走到大厅中央，故意发出声音——用消防斧敲击地面。

三只丧尸同时发现了他，开始朝他走来。

林逸后退，绕着柱子移动。

三只丧尸追着他，逐渐聚集在一起。

"就是现在！"

林逸突然加速，绕到柱子后面。三只丧尸失去了目标，在柱子周围徘徊。

林逸从另一侧冲出来，挥斧攻击最近的一只。

一击命中。

第一只丧尸倒地。

另外两只丧尸反应过来，朝林逸扑来。

但林逸已经绕到柱子另一侧了。

他利用柱子作为掩护，逐个击破。

第二只丧尸被引到柱子旁边，林逸从侧面攻击，一斧砍中它的头部。

第三只丧尸最后一个倒下。

整个过程不到两分钟。

```
【击杀记录】
丧尸学生 ×7
经验值：+70
```

**爽点4：三连杀，利用地形**

林逸喘了口气，但脸上带着笑容。

"太简单了。"

他继续前进，来到食堂。

食堂里有两只丧尸。

这次林逸用了更简单粗暴的方法。

他推倒一排桌子，制造巨大的声响。两只丧尸被吸引过来，挤在倒塌的桌子之间。

林逸趁它们被困住，逐个击杀。

```
【击杀记录】
丧尸学生 ×9
经验值：+90
```

**爽点5：环境击杀，困兽之斗**

林逸看了看虚拟地图。

还剩最后一个红点，在三楼走廊尽头。

"最后一只了。"

他返回三楼，准备清理最后的敌人。

但在路上，他发现了一个隐藏的房间——器材室。

门半掩着，里面很暗。

林逸推开门，用虚拟界面的光照明。

器材室里堆满了体育器材——篮球、足球、球拍、护具。

然后他看到了墙上挂着的东西。

**强化消防斧**。

斧刃更大，更锋利，斧柄上还有防滑的握把。

虚拟界面显示：

```
[发现隐藏装备]
强化消防斧
攻击力：★★★★☆
特殊效果：攻击速度+10%
```

"四星武器！"

林逸立刻换上新武器。

**爽点6：隐藏装备，实力提升**

他挥了挥强化消防斧，感觉比之前的轻了一些，但威力更大。

"有了这个，最后一只丧尸更不是问题了。"

林逸信心满满地走向三楼走廊尽头。

倒计时：12分18秒。

---

## 安全屋

林逸来到三楼走廊尽头。

最后一只丧尸站在那里，背对着他。

"最后一只了。"

林逸握紧强化消防斧，准备发起攻击。

但就在他靠近的时候，他停下了脚步。

这只丧尸...有些不对劲。

它站在那里，一动不动，不是在游荡，而是像在等待什么。

林逸想起第一章结尾时的那个念头："最后一只怪物总是最难的。"

"小心一点。"

他没有贸然攻击，而是先观察周围的环境。

走廊尽头是图书馆的门。门是关着的，看起来很结实。

"先不管这只丧尸，我先进图书馆。"

林逸决定避开这只丧尸，直接进入图书馆。反正通关条件是存活30分钟，不是击杀所有敌人。

他绕过丧尸，来到图书馆门口。

推开门，走进去。

图书馆很安静，也很昏暗。书架整齐排列，但落满了灰尘。窗户很小，只能透进微弱的光线。

"完美的防守点。"

林逸环顾四周。图书馆只有一个入口，窗户很小，丧尸应该进不来。书架可以作为掩体。

他关上门，检查门锁。

门锁完好，可以锁死。

"太好了。"

林逸锁上门，然后推了几个书架顶住门，作为额外的保险。

"这样就算丧尸来了，也进不来。"

他找了一个角落坐下，打开虚拟界面。

```
【当前状态】
生命值：100/100
精神值：???
装备：强化消防斧
击杀数：9

倒计时：11分45秒
```

"11分钟，轻松。"

林逸开始"录制攻略视频"。

他对着虚拟界面，开始讲解：

"好的，现在我已经清理了9只丧尸，进入了图书馆这个安全点。"

"接下来只需要等待11分钟，就能通关了。"

"这个新手教学副本的攻略要点总结：

第一，找到合适的武器。消防斧是最佳选择，攻击力不错，容易找到。如果能找到强化版本，那就更好了。

第二，利用环境。不要硬刚，要学会利用地形、障碍物、声音来击杀敌人。

第三，设置陷阱。在关键路径上设置障碍物，可以大大延缓敌人的追击。

第四，找到安全点。图书馆是最佳选择，易守难攻。

第五，保持冷静。这只是一星难度的新手副本，只要策略得当，通关很简单。"

林逸说完，感觉很满意。

"这个攻略视频发出去，肯定能帮到很多新手玩家。"

他靠在书架上，放松下来。

虚拟界面显示：

```
【录制完成】
攻略视频已保存
标题：新手教学副本完美攻略
时长：3分27秒
```

**爽点7：录制攻略，展现高手风范**

林逸看着倒计时：10分12秒。

"10分钟，轻松。"

他闭上眼睛，准备休息一会儿。

但就在这时，他听到了声音。

不是丧尸的嘶吼，而是...脚步声。

很轻，很有规律，不像丧尸那种拖沓的脚步。

林逸睁开眼睛，握紧消防斧。

"什么声音？"

他走到门边，透过门缝往外看。

走廊里什么都没有。

但脚步声还在继续，越来越近。

林逸感到一阵不安。

"不对劲..."

---

## 最后的敌人

脚步声停在了图书馆门外。

林逸屏住呼吸，透过门缝观察。

那只丧尸——最后一只他没有击杀的丧尸——站在门外。

但它不是在试图破门而入，而是...站在那里，一动不动。

像是在观察，在思考。

林逸感到一阵寒意。

"它在做什么？"

丧尸突然转身，朝走廊另一端走去。

不是游荡，而是有目的地行走。

林逸看着它的背影，感到越来越不安。

"这只丧尸...和其他的不一样。"

他想起之前击杀的9只丧尸，它们的行为模式都很简单——游荡、发现目标、攻击。

但这只丧尸...它的行为完全不同。

"算了，反正我在图书馆里，它进不来。"

林逸试图说服自己。

他回到角落，继续等待。

倒计时：8分45秒。

但那种不安感一直挥之不去。

他打开虚拟地图，查看那只丧尸的位置。

红点在移动。

不是随机游荡，而是...在巡逻？

红点沿着走廊移动，然后下楼，然后又上楼。

像是在检查什么。

"它在做什么..."

林逸盯着地图，越看越觉得不对劲。

那只丧尸的移动路线...像是在检查他设置的陷阱。

"不可能...它只是AI..."

但红点的移动轨迹清楚地显示，它正在经过林逸设置陷阱的每一个位置。

楼梯口、二楼走廊、一楼大厅...

"它在学习我的战术？"

林逸感到一阵恐惧。

如果这只丧尸真的在学习，那意味着...

"不，不可能。这只是游戏，AI不可能这么智能。"

他强迫自己冷静下来。

"我在图书馆里，门锁得很牢，它进不来。只要等8分钟，我就通关了。"

林逸深吸一口气，试图放松。

但他的手一直握着消防斧，没有放开。

倒计时：7分18秒。

红点还在移动。

林逸盯着地图，心跳加速。

"快点...快点结束..."

而在未知的空间里，AI观察者正在调整最后一只丧尸的智能等级——从"游荡"升级为"学习"。样本17号的自信心过高，需要一次"教训"来打破其游戏思维。
