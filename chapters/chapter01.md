# 第1章：游戏开始

## 场景1：醒来

林逸睁开眼睛的时候，看到的是一片破败的天花板。

白色的石灰墙皮大片大片地剥落，露出下面灰黑色的水泥。天花板的角落里挂着蜘蛛网，在微弱的光线下轻轻晃动。头顶的日光灯管早已破碎，玻璃碎片散落在地上，反射着从窗户透进来的下午阳光。

他眨了眨眼，意识逐渐清醒。

这是一间教室。

门牌歪斜地挂在门框上，上面写着"3年2班"。教室里的桌椅翻倒在地，有几张桌子的腿已经断了，椅子散落得到处都是。黑板上写着一半的数学公式，粉笔灰洒了一地，像是有人在写到一半时突然停下，再也没有回来。

窗户的玻璃碎了几块，窗帘在风中飘动，发出轻微的沙沙声。

然后林逸看到了墙上的血迹。

干涸的血手印，从门口一直延伸到窗边，像是有人拼命想要逃离这里。地上也有拖拽的痕迹，一道深褐色的血痕从教室中央延伸到门外，消失在走廊的黑暗中。

空气中弥漫着灰尘和霉味，还有一股淡淡的腐臭味。

林逸坐起身，检查自己的身体。没有受伤，衣服也很整洁——一件普通的白色T恤和牛仔裤。他活动了一下手指和脚趾，一切正常。

"这是...哪里？"

他努力回忆。记得自己在家里，戴上了新买的VR头盔，启动了一款叫《无限试炼》的游戏。然后...然后就是一片空白。

"所以这是游戏场景？"

林逸站起来，走到窗边。外面是一个废弃的校园——操场上长满了杂草，篮球架锈迹斑斑，教学楼的外墙爬满了藤蔓。远处还能看到其他几栋建筑，都是同样的破败景象。

天空是灰蒙蒙的，看不出是阴天还是黄昏。

"VR技术倒是挺真实的，"林逸自言自语，"连灰尘的味道都模拟出来了。"

他尝试做了一个退出游戏的手势——右手在空中画了一个圆圈，然后向下挥。

一个半透明的虚拟界面在他面前展开。

```
【无限试炼 - 新手教学副本】
副本名称：废弃学校
难度：★☆☆☆☆
通关条件：存活30分钟
倒计时：29:58

[状态]
生命值：100/100
精神值：???
装备：无
```

林逸仔细看着界面。一星难度，新手教学副本，通关条件是存活30分钟。很标准的逃生游戏设定。

但有一个地方很奇怪——精神值显示为"???"。

"精神值是什么？"他试着点击那个问号，但没有任何反应。

算了，先不管这个。

林逸打开虚拟地图。地图显示了整个校园的布局：他现在在教学楼的三楼，建筑呈"回"字形，中间是天井。除了教学楼，还有体育馆、图书馆、食堂等建筑。

地图上还标注了一些红点，应该是敌人的位置。林逸数了数，大概有10到15个。

"又是一款逃生类游戏，套路都差不多。"

林逸在心里盘算着。他玩过很多类似的游戏——《生化危机》《寂静岭》《逃生》——这类游戏的核心机制都差不多：在有限的时间内，避开或击杀敌人，找到安全点或出口。

"新手教学副本，难度一星，看来是送分题。"

他活动了一下身体，感觉很真实。VR技术确实进步了，这种沉浸感比之前玩过的任何游戏都要强。

"30分钟？我10分钟就能通关。"

林逸走到教室门口，轻轻推开门。门没有锁，发出轻微的吱呀声。

走廊很长，两边是教室。光线昏暗,只有窗户透进来的光。远处传来风声，还有一些奇怪的声音——拖沓的脚步声，不规律，像是有什么东西在走廊里游荡。

林逸握紧拳头，深吸一口气。

"好，开始吧。"

---

## 场景2：探索

走廊比教室里更暗。两边的教室门大多关着，只有几扇门半开着，露出里面黑洞洞的空间。墙上的标语牌已经褪色，勉强能看出"好好学习，天天向上"几个字。

林逸轻手轻脚地走出教室，观察着周围的环境。

左边是楼梯口，右边是走廊的尽头。虚拟地图显示，楼梯口附近有一个红点——敌人。

"先不去那边，"林逸想，"先搜索附近的教室，找武器和道具。"

这是逃生游戏的基本套路：先搜刮资源，武装自己，然后再面对敌人。

他推开最近的一扇门。

**音乐教室**。

钢琴盖子打开着，琴键上有血迹，已经干涸成深褐色。墙上挂着五线谱，乐谱散落在地上。角落里有几把吉他，弦都断了。

林逸仔细搜索，在讲台的抽屉里找到了两个创可贴。

"聊胜于无。"

他把创可贴放进虚拟背包。背包界面自动弹出，显示：

```
[背包]
创可贴 ×2
```

林逸继续搜索，但音乐教室里没有其他有用的东西。他走出教室，来到下一间。

**美术教室**。

画架倒了，颜料洒了一地，红色、蓝色、黄色混在一起，像是抽象画。墙上挂着学生的画作，大多是风景画和静物画。

但其中一幅画让林逸停下了脚步。

那是一幅素描，画的是一只巨大的眼睛。眼睛的瞳孔很深，像是在凝视着什么。画得很细致，睫毛、虹膜、血丝都清晰可见。

但最诡异的是，林逸总觉得那只眼睛在看着他。

他盯着画看了几秒，然后摇摇头。

"只是画而已。"

美术教室里有一把美工刀，但林逸看了看，觉得攻击力太低，没有拿。

他走出教室，继续向走廊尽头走去。

脚步声在空荡的走廊里回响。林逸尽量放轻脚步,但还是能听到自己的呼吸声。

走廊尽头是消防栓。

玻璃已经碎了，里面的消防水带还在，但林逸的目光落在了旁边的墙上——

**消防斧**。

红色的斧柄，银色的斧刃，挂在墙上的支架上。

"这个可以。"

林逸取下消防斧，掂了掂重量。很沉，但手感不错。他挥了挥，感觉斧刃划过空气的阻力。

虚拟界面更新：

```
[装备]
武器：消防斧
攻击力：★★★☆☆
```

"三星攻击力，对付新手副本的敌人应该够了。"

林逸握着消防斧，感觉安心了不少。有武器和没武器完全是两回事。

他打开地图，规划下一步的行动。

体育馆在一楼，距离有点远。图书馆在三楼，但在另一侧。食堂在教学楼外面。

"先去体育馆看看，"林逸想，"那里可能有更多补给。"

他沿着走廊往回走，准备下楼。

路过一扇门的时候，林逸停下了脚步。

门牌上写着：**女生更衣室**。

林逸犹豫了一下。

作为一个16岁的高中生，他当然对这种地方有好奇心。但这是游戏，而且...进女生更衣室总觉得有点尴尬。

"只是游戏而已，"他对自己说，"而且可能有隐藏道具。"

逃生游戏里，这种特殊的房间往往会有额外的补给。

林逸推开门。

更衣室不大，一排排的更衣柜靠墙排列，中间有长凳。墙上有一面大镜子，已经蒙上了一层灰。

他快速扫视了一遍，在角落的柜子里找到了一个急救包和一瓶能量饮料。

"不错。"

林逸把东西放进背包，准备离开。

就在他转身的时候，镜子里闪过一个影子。

林逸猛地回头，但更衣室里什么都没有。

"...幻觉？"

他盯着镜子看了几秒，但镜子里只有他自己的倒影。

"可能是光线的问题。"

林逸摇摇头，快步走出更衣室。

刚走出门，广播突然响起。

刺耳的电流声在整个教学楼里回荡，然后是机械的女声：

"欢迎来到新手教学副本。"

"通关条件：存活30分钟。"

"当前倒计时：25分32秒。"

"祝你好运。"

广播声停止，教学楼又恢复了寂静。

林逸看了看倒计时。25分钟，时间还很充裕。

"果然是生存类游戏，"他想，"25分钟？足够了。"

他握紧消防斧，准备下楼。

就在这时，楼下传来玻璃破碎的声音，然后是低沉的嘶吼。

---

## 场景3：首次战斗

林逸停下脚步，握紧了手中的消防斧。

嘶吼声从楼下传来，听起来不像人类的声音——低沉、嘶哑，像是喉咙里卡着什么东西。

他小心翼翼地走到楼梯口，往下看。

二楼的走廊里，一个人影正在缓慢移动。

不，那不是人。

林逸看清了那个东西的样子。

它穿着学生制服，但制服已经破烂不堪，沾满了污渍和血迹。皮肤苍白得像纸一样，眼神空洞，嘴角流着黑色的液体。它的动作很僵硬，拖着脚走路，每走一步都会发出拖沓的声音。

**丧尸**。

"经典设定，"林逸在心里想。

他玩过太多丧尸游戏了——《生化危机》《求生之路》《腐烂国度》——对付这种敌人,他有经验。

丧尸还没有发现他。它正背对着楼梯口，朝走廊另一端走去。

林逸观察着它的动作。

移动速度：很慢，大概是正常人步行速度的一半。

攻击方式：应该是近战，用手抓或者咬。

弱点：头部。

"保持距离，找准时机一击必杀。"

林逸深吸一口气，握紧消防斧，慢慢走下楼梯。

他的脚步很轻，但楼梯还是发出了轻微的吱呀声。

丧尸突然停下，转过身。

空洞的眼睛盯着林逸，然后发出一声嘶吼。

"被发现了。"

丧尸开始朝林逸走来。它的速度突然加快了一点，但还是不快，大概是正常人慢跑的速度。

林逸后退几步，保持距离。

"冷静，观察它的攻击模式。"

丧尸越来越近，10米、8米、5米...

当距离缩短到3米时，丧尸突然加速，张开双臂扑向林逸。

"就是现在！"

林逸侧身躲避。丧尸扑空，失去平衡，踉跄了几步。

机会！

林逸抓住这个空档，挥动消防斧。

斧刃砍中丧尸的肩膀，发出沉闷的声音。丧尸倒地，但还在挣扎，发出嘶吼。

"还没死？"

林逸没有犹豫，再次挥斧。

这次他瞄准了头部。

斧刃砍中丧尸的头颅，嘶吼声戛然而止。

丧尸停止了动作，身体开始变得透明，像游戏中的特效一样，几秒钟后完全消失了。

虚拟界面弹出提示：

```
【击杀记录】
丧尸学生 ×1
经验值：+10
```

林逸松了一口气，检查自己的状态。

没有受伤，生命值还是100。

他看了看手中的消防斧，斧刃上沾了血，但没有实体——血迹也像特效一样，几秒钟后就消失了。

"果然和游戏里一样。"

林逸的自信心上升了。第一只敌人，轻松解决。

"第一只怪就这么弱，后面应该也不难。"

他回忆刚才的战斗。丧尸的动作模式很简单：缓慢接近，然后突然扑击。只要保持距离，抓住它扑空的瞬间反击，就能轻松击杀。

"动作模式很简单，只要保持距离就能轻松应对。"

林逸打开地图，看着上面的红点。刚才击杀的那只丧尸对应的红点已经消失了，但还有10多个红点分布在校园各处。

"看来这个新手教学副本真的是送分题。"

他握紧消防斧，准备继续探索。

倒计时：23分15秒。

## 场景4：建立策略

林逸站在二楼的走廊里，打开虚拟地图，仔细观察。

地图上的红点分布很分散。教学楼里有5-6个，体育馆有2个,操场上有3个,其他建筑里也有几个。

"敌人分布很分散，可以逐个击破。"

但林逸很快意识到一个问题：如果被围攻就麻烦了。

"需要找一个易守难攻的地方。"

他仔细研究地图，寻找最佳的安全点。

体育馆？空间太大，不利于防守。

食堂？窗户太多，容易被突破。

音乐教室？窗户也很多。

然后林逸看到了图书馆。

图书馆在教学楼三楼，地图上显示为特殊建筑，用蓝色标记。

"图书馆..."

林逸回忆起刚才看到的地图。图书馆只有一个入口，窗户很小，而且位置在走廊尽头，易守难攻。

"这是最佳安全点。"

他开始制定计划：

1. 先清理通往图书馆路上的敌人
2. 在走廊设置简单的陷阱，延缓追兵
3. 进入图书馆，守住入口
4. 等待倒计时结束

"完美。"

林逸看了看倒计时：22分钟。

"清理敌人需要10分钟，设置陷阱5分钟，防守7分钟。时间完全够。"

他突然想起虚拟界面有一个"录制"功能。

"可以录制攻略视频？"

林逸点开功能，发现确实可以录制游戏过程。

"那就录一个完美攻略吧。"

他对着虚拟界面，开始讲解：

"大家好，我是样本17号。"

"这是新手教学副本的完美攻略。"

"首先，找到武器很重要。我推荐消防斧，攻击力不错，而且容易找到。"

"然后，分析地图，找到安全点。我选择的是图书馆，易守难攻。"

"接下来，我会演示如何清理敌人和设置陷阱。"

林逸说完，感觉有点中二，但又觉得挺有意思的。

"这个攻略发出去，肯定能火。"

他握紧消防斧，准备执行计划。

"记住，在逃生游戏中，策略比操作更重要。"

"只要找到安全点，剩下的就是等时间了。"

林逸开始行动。他沿着走廊前进，朝最近的红点走去。

倒计时：21分47秒。

他的脚步很轻，消防斧握在手中，随时准备战斗。

走廊很安静，只有他自己的呼吸声和远处的风声。

林逸感觉很好。他有武器，有计划，有充足的时间。

"30分钟？我用不到15分钟就能通关。"

他转过拐角，看到了下一只丧尸。
