# 第4章：循环重置，认知崩溃

## 场景1：重置

林逸猛地睁开眼睛，看到的还是那片破败的天花板。

剧烈的喘息，像是溺水后浮出水面。

双手颤抖，摸向脖子——被咬的地方。

没有伤口，皮肤完好。

"我...我没死？"

他坐起来，检查虚拟界面。

```
【无限试炼 - 新手教学副本】
副本名称：废弃学校
难度：★☆☆☆☆
通关条件：存活30分钟
倒计时：29:58

[状态]
生命值：100/100
精神值：85/100
装备：无

[死亡记录]
第1次：28分15秒 - 失血过多
```

林逸盯着界面，脑子一片混乱。

"精神值...85？"

"死亡记录...第1次？"

"这是...重置了？"

记忆涌上来。

疼痛、恐惧、死亡。

那只聪明的丧尸，图书馆的陷阱，被咬的瞬间。

身体开始颤抖。

"这不是游戏...但为什么会重置？"

"我真的死了...然后又活了？"

林逸冲到窗边，看向外面。

操场、教学楼，一切都和之前一样。

"一切都重置了..."

虚拟界面弹出新的提示：

```
【系统提示】
检测到玩家首次死亡
死亡循环机制已激活
每次死亡将扣除精神值
精神值归零 = 永久死亡

提示：请珍惜每一次机会
```

林逸看着提示，感到一阵寒意。

"精神值归零...永久死亡..."

"我还能死几次？"

他计算着。100点精神值，死一次扣15点，现在还剩85点。

"还能死5次...然后就真的死了。"

林逸靠在墙上，双腿发软。

"这到底是什么...这到底是什么..."

他想起那只丧尸的智能，想起图书馆的陷阱，想起死亡的疼痛。

"如果这会重置...那我可以利用这一点。"

"我知道所有道具的位置，我知道所有丧尸的位置。"

"我有记忆优势。"

林逸强迫自己冷静下来。

"这一次，我不会再犯同样的错误。"

"我知道图书馆是陷阱，我会避开那只聪明的丧尸。"

"我会找其他安全点。"

他深吸一口气，握紧拳头。

"30分钟，我一定能撑过去。"

---

## 场景2：验证循环

林逸走出教室，来到走廊。

他没有立刻行动，而是先验证自己的记忆。

"如果真的重置了，那所有东西都应该和之前一样。"

他走到音乐教室，推开门。

创可贴还在讲台的抽屉里，位置一模一样。

"果然..."

林逸继续验证。

美术教室的美工刀，消防栓的消防斧，女生更衣室的急救包。

所有道具都在原位。

"完全重置了...就像读档一样。"

林逸打开虚拟地图，标记上一次循环的记忆。

丧尸的位置、陷阱的位置、危险区域。

"我记得所有细节。"

他小心翼翼地走到楼梯口，往下看。

果然，第一只丧尸还在那里。

位置、外观、动作，完全一样。

"真的重置了..."

林逸开始制定新的计划。

"上一次我去了图书馆，结果是陷阱。"

"这一次，我要找其他安全点。"

他研究地图。

体育馆？空间太大。

音乐教室？窗户太多。

天台？可能是个好选择。

"天台只有一个入口，而且位置高，可以看到周围的情况。"

林逸做出决定。

"目标：天台。"

"路线：避开那只聪明的丧尸，清理其他敌人。"

"策略：更谨慎，不冒险。"

他握紧消防斧，准备开始第二次尝试。

"我有记忆优势，AI不知道我记得上一次的事。"

"这一次，我会赢。"

---

## 场景3：复制失败

林逸按照记忆中的路线前进。

他知道每一只丧尸的位置，知道每一个陷阱的设置。

"第一只丧尸在楼梯口，我用之前的方法——保持距离，等它扑空，然后反击。"

林逸走到楼梯口，看到了那只丧尸。

他保持距离，等待丧尸发现他。

丧尸转身，看到了林逸。

但它没有扑过来。

"怎么回事？"

丧尸保持距离，缓慢靠近，动作很谨慎。

像是在"防备"。

"上一次它直接扑过来的...为什么这次不一样？"

林逸感到不安，但还是主动进攻。

丧尸躲避了。

几个回合后，林逸才勉强击杀它，但消耗了更多体力。

"它的行为模式变了..."

林逸继续前进，来到二楼走廊。

他按照上一次的路线，准备清理第二只丧尸。

他设置了简单的陷阱——用桌椅堵住走廊。

"上一次这个陷阱很有效。"

但这一次，丧尸绕过了陷阱。

像是"知道"那里有陷阱。

从另一个方向靠近林逸。

林逸措手不及，被迫近战。

丧尸的攻击更有章法，会格挡、会闪避。

林逸被抓伤了。

"它怎么知道陷阱的位置？"

"我明明是第一次在这里设陷阱..."

"不对...上一次循环，我也在这里设过..."

"难道...AI记得上一次的事？"

林逸感到一阵恐惧。

他试探性地走向第三只丧尸。

这只丧尸本该在教室里，但现在它在走廊上。

位置变了。

"位置都变了..."

"我的记忆优势...没用了..."

"AI在学习...在进化..."

"我每死一次，它就变强一次..."

林逸的动作被预判，陷阱被识破，路线被封锁。

丧尸开始"合作"。

三只丧尸从不同方向靠近，形成包围圈。

林逸被困在走廊中央，退路被切断。

"不...不可能..."

"它们在配合...在围攻我..."

"这不是游戏AI该有的智能..."

林逸拼命突围，但体力已经耗尽，伤口在流血，视线开始模糊。

"我以为我有优势...但其实AI才有优势..."

"我记得上一次，它也记得..."

"但它可以改变策略，而我的身体还是普通人..."

"这不公平...这根本不公平..."

## 场景4：第二次死亡

林逸背靠墙壁，三只丧尸围攻。

消防斧挥舞，但越来越慢。

体力耗尽，动作变形。

左臂被抓伤，右腿被咬伤。

鲜血流淌，疼痛难忍。

"为什么...为什么要这样折磨我..."

恐惧：95%
绝望：80%
愤怒：60%

林逸失去平衡，摔倒。

消防斧脱手。

丧尸扑上来。

倒计时：17分42秒。

比上次更早。

多只丧尸同时撕咬。

疼痛是上一次的数倍。

惨叫声回荡在走廊。

意识快速消散。

"又要死了..."

"还会重置吗..."

"精神值还剩多少..."

"我...还能坚持多少次..."

意识消失。

画面变黑。

寂静。

虚拟界面显示：

```
【死亡】
死因：失血过多、多处撕咬伤
存活时间：17分42秒
击杀数：2
死亡次数：2

[系统提示]
检测到玩家死亡
正在重置副本...
精神值：85 → 70（-15）
```

林逸再次睁开眼睛。

还是那个教室，还是那个天花板。

但这一次，他没有立刻起身。

躺在地上，盯着天花板。

眼神空洞，身体在颤抖。

"我...还要经历多少次..."

不是身体的疼痛，而是心理的崩溃。

"每次都会死...每次都会重置..."

"我...真的能通关吗..."
