# 第1章场景卡片：游戏开始

**章节目标**：建立世界观和基本规则，展现主角的游戏高手身份，制造"这很简单"的假象，埋下第一批爽点

**目标字数**：5,000字
**节奏**：慢→中→快

---

## 场景1：醒来（1,000字）

### 场景目标
- 建立废弃学校的氛围
- 展现主角的冷静理性
- 引入虚拟界面和基本规则
- 让读者理解"这是一个游戏"

### 开场画面
**第一句话**：林逸睁开眼睛的时候，看到的是一片破败的天花板。

**环境细节**：
- 教室：3年2班的门牌歪斜着挂在门框上
- 桌椅：翻倒在地，腿断了几根
- 黑板：写着一半的数学公式，粉笔灰洒了一地
- 窗户：玻璃碎了几块，窗帘在风中飘动
- 血迹：墙上有干涸的血手印，地上有拖拽的痕迹
- 光线：下午的阳光斜射进来，照出空气中的灰尘

### 主角反应
1. **第一反应**：冷静观察环境（不是惊慌）
2. **第二反应**：检查自己的身体（没有受伤）
3. **第三反应**：尝试回忆（记得玩了一款VR游戏《无限试炼》）
4. **第四反应**：判断这是游戏场景

### 虚拟界面出现
**触发**：林逸尝试"退出游戏"的手势

**界面内容**：
```
【无限试炼 - 新手教学副本】
副本名称：废弃学校
难度：★☆☆☆☆
通关条件：存活30分钟
倒计时：29:58

[状态]
生命值：100/100
精神值：???
装备：无
```

### 主角心理
- **初始想法**："又是一款逃生类游戏，套路都差不多。"
- **游戏经验**：回忆起玩过的类似游戏（《生化危机》《寂静岭》）
- **策略思维**：开始分析环境，寻找武器和出口
- **情绪**：轻松、自信，甚至有点期待

### 关键动作
1. 站起来，活动身体
2. 走到窗边，观察外面（操场、教学楼）
3. 尝试打开教室门（门没锁）
4. 检查虚拟界面的功能（地图、状态、背包）

### 关键对话（内心独白）
- "VR技术倒是挺真实的，连灰尘的味道都模拟出来了。"
- "新手教学副本，难度一星，看来是送分题。"
- "30分钟？我10分钟就能通关。"

### 伏笔/细节
- 血手印的位置（暗示有人试图逃离）
- 黑板上的数学公式（后期可能是线索）
- 虚拟界面中"精神值"显示为"???"（暗示这个系统的存在）
- 林逸的网名在界面上显示为"样本17号"（暗示他不是第一个）

### 转折点
听到走廊传来脚步声（拖沓、不规律）

## 场景2：探索（1,500字）

### 场景目标
- 展现主角的游戏思维
- 找到第一件武器（消防斧）
- 埋下性好奇的细节
- 建立"这很简单"的感觉

### 场景分段

#### 2.1 走廊探索（500字）
**环境**：
- 走廊：长长的走廊，两边是教室
- 光线：昏暗，只有窗户透进来的光
- 声音：风声、远处的脚步声、自己的呼吸声

**主角行动**：
- 轻手轻脚地走出教室
- 观察走廊两端（左边是楼梯，右边是尽头）
- 打开虚拟地图（显示当前位置和建筑结构）
- 标记"安全点"和"危险区域"

**游戏思维体现**：
- "走廊太开阔，不利于防守，得找个房间。"
- "楼梯口通常是关键点，可能有补给。"
- "先搜索附近的教室，找武器和道具。"

#### 2.2 搜索教室（600字）
**第一间教室**：音乐教室
- 钢琴盖子打开，琴键上有血迹
- 找到：创可贴×2（放入背包）
- 没有武器

**第二间教室**：美术教室
- 画架倒了，颜料洒了一地
- 找到：美工刀（攻击力太低，放弃）
- 墙上有学生的画作（其中一幅画的是"观察者"的眼睛？）

**走廊尽头**：消防栓
- 玻璃已经碎了
- 找到：**消防斧**（第一件真正的武器！）
- 林逸挥了挥，感觉重量和手感
- "这个可以。"

**爽点**：找到武器的成就感

#### 2.3 女生更衣室（400字）
**位置**：体育馆旁边

**主角心理**：
- 路过时看到门牌
- 犹豫了一下（青春期的好奇）
- 想起这是"游戏"，可能有隐藏道具
- 但又觉得有点尴尬

**行动**：
- 推开门，快速扫视
- 环境：更衣柜、长凳、镜子
- 找到：急救包×1、能量饮料×1
- 没有多停留，快速离开

**心理描写**：
- "只是游戏而已，没什么好尴尬的。"
- 但还是有点脸红（真实的青春期反应）

**伏笔**：
- 镜子里一闪而过的影子（是幻觉？还是有什么？）

### 关键对话（内心独白）
- "消防斧，近战武器，攻击力应该不错。"
- "急救包和能量饮料，标准配置。"
- "这游戏的道具分布挺合理的，符合逃生游戏的套路。"

### 广播响起
**时机**：林逸离开更衣室后

**内容**：
- 刺耳的电流声
- 机械的女声："欢迎来到新手教学副本。"
- "通关条件：存活30分钟。"
- "当前倒计时：25分32秒。"
- "祝你好运。"

**主角反应**：
- "果然是生存类游戏。"
- "25分钟？足够了。"

### 转折点
听到楼下传来玻璃破碎的声音，然后是低沉的嘶吼

## 场景3：首次战斗（1,500字）

### 场景目标
- 展现主角的战斗能力
- 建立丧尸的威胁感（但不太强）
- 制造爽点（轻松击杀）
- 让主角更加自信

### 场景分段

#### 3.1 遭遇（400字）
**位置**：楼梯口

**丧尸出现**：
- 外观：穿着学生制服，脸色苍白，眼神空洞
- 动作：僵硬、缓慢，拖着脚走
- 声音：喉咙里发出低沉的嘶吼
- 距离：10米

**主角反应**：
- 第一反应：握紧消防斧
- 第二反应：观察敌人（速度、攻击方式）
- 第三反应：回忆FPS游戏经验
- 情绪：紧张，但不恐慌

**游戏思维**：
- "移动速度慢，攻击方式应该是近战。"
- "保持距离，找准时机一击必杀。"
- "头部应该是弱点。"

#### 3.2 战斗（700字）
**第一回合**：试探
- 丧尸发现林逸，开始靠近
- 林逸后退，保持距离
- 丧尸突然加速（但还是不快）
- 林逸侧身躲避

**第二回合**：反击
- 丧尸扑空，失去平衡
- 林逸抓住机会，挥斧
- 斧刃砍中丧尸的肩膀
- 丧尸倒地，但还在挣扎

**第三回合**：补刀
- 林逸再次挥斧
- 这次瞄准头部
- 丧尸停止动作
- 身体开始消散（像游戏中的特效）

**战斗细节**：
- 林逸的动作：利落、精准（FPS经验的体现）
- 丧尸的反应：机械、可预测
- 环境利用：利用楼梯的高度差

**爽点**：
- 成功击杀的成就感
- "果然和游戏里一样"的验证感
- 战斗奖励：经验值+10（虚拟界面显示）

#### 3.3 战后（400字）
**主角反应**：
- 松了一口气
- 检查自己（没有受伤）
- 检查消防斧（沾了血，但没有实体）
- 观察丧尸消失的地方（什么都没留下）

**心理变化**：
- 从紧张到放松
- 自信心上升
- "这游戏也就这样"

**关键对话（内心独白）**：
- "第一只怪就这么弱，后面应该也不难。"
- "动作模式很简单,只要保持距离就能轻松应对。"
- "看来这个新手教学副本真的是送分题。"

**虚拟界面更新**：
```
【击杀记录】
丧尸学生 ×1
经验值：+10
```

### 转折点
虚拟地图上出现了更多红点（敌人标记），分布在整个教学楼

## 场景4：建立策略（1,000字）

### 场景目标
- 展现主角的策略思维
- 找到"安全屋"（图书馆）
- 制定"完美攻略"
- 开始"录制攻略视频"（中二感）
- 为第2章做铺垫

### 场景分段

#### 4.1 分析局势（300字）
**主角行动**：
- 打开虚拟地图，仔细观察
- 标记敌人位置（10-15个红点）
- 标记自己的位置（蓝点）
- 寻找"安全点"

**游戏思维**：
- "敌人分布很分散，可以逐个击破。"
- "但如果被围攻就麻烦了。"
- "需要找一个易守难攻的地方。"

**发现**：
- 图书馆（地图上显示为特殊建筑）
- 位置：教学楼三楼
- 特点：只有一个入口，窗户很小
- 判断："这是最佳安全点。"

#### 4.2 制定计划（400字）
**策略**：
1. 先清理通往图书馆路上的敌人
2. 在走廊设置简单的陷阱（延缓追兵）
3. 进入图书馆，守住入口
4. 等待倒计时结束

**陷阱设计**：
- 用桌椅堵住部分走廊
- 在楼梯口放置障碍物
- 利用教室门制造"瓶颈"

**时间规划**：
- 当前倒计时：22分钟
- 清理敌人：10分钟
- 设置陷阱：5分钟
- 防守：7分钟
- "完美。"

#### 4.3 "录制攻略"（300字）
**主角行动**：
- 发现虚拟界面有"录制"功能
- 决定录制一个攻略视频
- 对着界面开始讲解

**攻略内容**：
- "大家好，我是样本17号。"
- "这是新手教学副本的完美攻略。"
- "首先，找到武器很重要，推荐消防斧。"
- "然后，分析地图，找到安全点。"
- "我选择的是图书馆，易守难攻。"
- "接下来，我会演示如何清理敌人和设置陷阱。"

**心态**：
- 得意、自信
- 有点中二（但符合16岁少年的性格）
- 完全把这当成游戏

**爽点**：
- "攻略主播"的优越感
- 对自己策略的自信
- 读者会觉得"这孩子要吃亏了"（反差感）

### 关键对话
**内心独白**：
- "这个攻略发出去，肯定能火。"
- "新手教学副本，我用不到15分钟就能通关。"

**对着界面说**：
- "记住，在逃生游戏中，策略比操作更重要。"
- "只要找到安全点，剩下的就是等时间了。"

### 章节结尾

**行动**：
- 林逸开始执行计划
- 握紧消防斧，走向最近的敌人
- 倒计时：21分47秒

**氛围**：
- 主角：自信满满
- 读者：期待反转

**AI视角插入**：
```
【观察者日志 - 样本17号】
初始反应：冷静、理性、过度自信
情感数据：好奇65%、兴奋42%、恐惧8%
战斗能力：良好（评分7/10）
策略思维：优秀（评分8.5/10）
评价：典型的游戏玩家思维，将真实当作虚拟
      过度依赖游戏经验，缺乏对未知的敬畏
预测：第一次死亡将在15-20分钟内发生
调整：暂不干预，观察其自然表现
备注：样本17号展现出超出平均水平的适应能力
      [老熊]基因标记检测中...进度15%
```

## 写作要点总结

### 节奏控制
- 场景1：慢（建立氛围）
- 场景2：中（探索发现）
- 场景3：快（战斗爽点）
- 场景4：中（策略规划）

### 情感曲线
- 开始：冷静、好奇
- 探索：兴奋、期待
- 战斗：紧张→放松→自信
- 结尾：得意、自满

### 爽点分布
1. 找到消防斧
2. 找到急救包和能量饮料
3. 成功击杀第一只丧尸
4. 发现"完美策略"
5. "录制攻略"的优越感

### 伏笔埋设
1. 精神值显示为"???"
2. "样本17号"的称呼
3. 镜子里的影子
4. AI日志中的"[老熊]基因标记"
5. 虚拟地图上的敌人分布（为第2章铺垫）

### 性格体现
- 冷静理性：面对未知环境不慌张
- 游戏思维：用游戏经验分析局势
- 社恐：在更衣室的尴尬
- 性好奇：对更衣室的犹豫
- 中二：录制攻略视频
